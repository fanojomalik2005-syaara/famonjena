CSS

CSS stands for Cascading Style Sheets. It is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS defines how elements should be displayed on screen, on paper, or in other media.

Basic Syntax

CSS is made up of rules that consist of a selector and a declaration block. The selector points to the HTML element you want to style, and the declaration block contains one or more declarations separated by semicolons. Each declaration includes a property and a value.

selector {
    property: value;
}

Example

Here is an example of a CSS rule that changes the color of all paragraphs to blue:

 
        

            

                
                p {
                    color: blue;
                }
